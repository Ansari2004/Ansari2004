## 🚀 What I'm Building

### SafeTrip — AI-Powered Travel Safety App
My final year project. SafeTrip generates **personalized city safety scores** using ML models, live air quality & weather APIs, and user health profiles.

- 🤖 7 ML models (XGBoost, LightGBM, CatBoost, Random Forest, Gradient Boosting, Ridge, MLP)
- 🌍 Real-time AQI & weather data via Open-Meteo & OpenWeatherMap APIs
- ⚡ FastAPI backend serving live predictions
- 📱 Mobile frontend (AI-assisted)
